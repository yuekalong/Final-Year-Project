# PAH2003 - Final Year Project

## About This Project

This is the final year project of the Chinese University of Hong Kong students who graduate at **2020 - 2021** and studied in **Computer Science**. The supervisor of this project is **Prof. HENG Pheng Ann**. And this is an open topic project. We shall work in one of the following areas - Virtual Reality, Augmented Reality, Visualization, Computer Graphics and Human Computer Interaction.

And we decided to make a game for this project for representing what we learn in this four years.
